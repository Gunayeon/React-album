# my-album📷
## 1. 소개
### 프로젝트 명 
> 🌄 사진 api를 활용한 이미지 검색 웹 사이트 🌉

### 프로젝트 내용
> 여러 종류의 사진을 제공하는 이미지 API를 활용해 **TypeScript** 기반 **React**로 개발된 웹사이트입니다. 사진과 정보를 제공할 뿐만 아니라 검색, 북마크 등 다양한 기능을 지원합니다.
